## Troubleshooting

### "Register button doesn't work"
- Make sure mobile is 10 digits
- Make sure passwords match
- Check browser console for errors (F12)

### "User not found" on login
- Use mobile number you registered with
- Check for typos
- Use `debugAuthSystem()` to verify mobile is registered

### "Data disappeared"
- Check localStorage isn't cleared
- Run: `localStorage.getItem('earnhubUsers')`
- Should show your user data

### "Still seeing login screen after register"
- Check browser console
- Look for red error messages
- Run `debugAuthSystem()` to diagnose
